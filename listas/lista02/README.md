## Como Compilar

**OBS!** Via gcc tanto no windows como no linux/macos

### Linux/MacOS
- Para compilar o código:
```bash
gcc questao.c -o ./builds/nome_do_build
```
- Para rodar o código:
```bash
./builds/nome_do_build
```
### Windows (via gcc MINGW)
- Para compilar o código:
```bash
gcc questao.c -o .\builds\nome_do_build
```
- Para rodar o código:
```bash
.\builds\nome_do_build.exe
```
