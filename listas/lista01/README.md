## Como Compilar

### Linux
- Para compilar o código:
```bash
gcc questao.c -o nome_do_build
```
- Para rodar o código:
```bash
./nome_do_build
```
### Windows (via gcc MINGW)
- Para compilar o código:
```bash
gcc questao.c -o nome_do_build
```
- Para rodar o código:
```bash
.\nome_do_build.exe
```
