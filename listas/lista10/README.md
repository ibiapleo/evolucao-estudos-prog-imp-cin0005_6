# Lista 10 — Estruturas de Dados (Parte 1: Pilhas, Filas e Listas Encadeadas)

## 📚 Conteúdo Abordado
* Implementação de Pilhas (LIFO) com Alocação Dinâmica de Memória
* Implementação de Filas (FIFO) com Alocação Dinâmica de Memória e Manipulação de Ponteiros e `structs`
* Conceitos de Listas Encadeadas

## 🧠 Aprendizados
> Prática na alocação dinâmica e implementação do zero de estruturas de dados fundamentais (pilhas, filas e listas encadeadas) na linguagem C, consolidando conceitos de gerenciamento de memória e manipulação de nós.

*Nota de Autoria: Todas as resoluções desta lista foram desenvolvidas de forma autoral, baseadas inteiramente em conhecimentos de algoritmos e estruturas de dados, sem a utilização de Inteligência Artificial para a escrita dos códigos.*

## 💻 Exercícios

| Arquivo | Estrutura | Descrição |
|---------|-----------|-----------|
| `q1.c`  | Pilha     | **Attack the Stack**: Implementação de lógica e controlo de fluxo utilizando pilhas. |
| `q2.c`  | Fila      | **Gerenciador de Fila de Tarefas**: Implementação de fila contendo `structs` personalizadas como Nós. |
| `q3.c`  | Fila      | **Novo Sistema de Matrícula SIGAA**: Aplicação prática de filas, também baseada em nós personalizados, para gerir matrículas. |

## 🚀 Como Compilar e Executar

### Linux/MacOS
```bash
gcc -Wall -Wextra -std=c11 q1.c -o q1
gcc -Wall -Wextra -std=c11 q2.c -o q2
gcc -Wall -Wextra -std=c11 q3.c -o q3
gcc -Wall -Wextra -std=c11 q4.c -o q4
./q4
```

### Windows (via gcc MINGW)
```bash
gcc -Wall -Wextra -std=c11 q1.c -o q1.exe
gcc -Wall -Wextra -std=c11 q2.c -o q2.exe
gcc -Wall -Wextra -std=c11 q3.c -o q3.exe
gcc -Wall -Wextra -std=c11 q4.c -o q4.exe
.\q4.exe
```
