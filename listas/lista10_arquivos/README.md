# Lista 10 — Arquivos

## Conteúdo Abordado

- Abertura e fechamento de arquivos: `fopen`, `fclose`
- Leitura e escrita em arquivos texto: `fprintf`, `fscanf`, `fgets`, `fputs`
- Leitura e escrita binária: `fread`, `fwrite`
- Modos de abertura: `"r"`, `"w"`, `"a"`, `"rb"`, `"wb"`
- Tratamento de erros com `ferror` e `feof`
